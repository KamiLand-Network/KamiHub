# 安装配置

> KamiHub 快速部署指南

本页面将指导您完成KamiHub插件的安装、配置和初始化设置，确保插件能够在您的服务器上正常运行。

## 系统要求

### 服务器环境
- **Minecraft版本**：1.21.4+
- **服务端类型**：Paper, Purpur（不兼容Bukkit和Spigot！）
- **Java版本**：Java 21+

### 依赖插件
- **必需依赖**：无
- **可选依赖**：
  - PlaceholderAPI（占位符支持）
  - LuckPerms（权限组识别）
  - Vault（权限组识别候补方案）

## 下载安装

### 1. 下载插件

[GitHub](https://github.com/KamiLand-Network/KamiHub/releases) | [BuiltByBit](https://www.builtbybit.com/resources/kamihub.1312/) | [Hangar](https://hangar.papermc.io/KamiLand-Network/KamiHub)

### 2. 安装插件

将下载好的插件文件（如`KamiHub-1.0.jar`）放入服务器目录下plugins文件夹内。
重启，然后输入`/plugins`检查插件状态。
