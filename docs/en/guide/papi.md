# PlaceholderAPI

> PlaceholderAPI (PAPI) Usage in KamiHub

## Usage Instructions

In most displayable text in `config.yml`, as well as all values in `messages.yml`, you can use PlaceholderAPI placeholders.

When these actions are triggered by players, the plugin will parse these variables as that player, ensuring placeholders are correctly replaced with the player's relevant information.

**Note**: Built-in KamiHub PAPI placeholders are expected to be added in version 1.1 ~ 1.2, providing more customization features.
