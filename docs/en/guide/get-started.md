# Installation and Configuration

> KamiHub Quick Deployment Guide

This page will guide you through the installation, configuration and initial setup of the KamiHub plugin to ensure the plugin can run normally on your server.

## System Requirements

### Server Environment
- **Minecraft Version**: 1.21.4+
- **Server Type**: Paper, Purpur (Not compatible with Bukkit and Spigot!)
- **Java Version**: Java 21+

### Plugin Dependencies
- **Required Dependencies**: None
- **Optional Dependencies**:
  - PlaceholderAPI (placeholder support)
  - LuckPerms (permission group identification)
  - Vault (permission group identification backup solution)

## Download and Installation

### 1. Download Plugin

[GitHub](https://github.com/KamiLand-Network/KamiHub/releases) | [BuiltByBit](https://www.builtbybit.com/resources/kamihub.1312/) | [Hangar](https://hangar.papermc.io/KamiLand-Network/KamiHub)

### 2. Install Plugin

Place the downloaded plugin file (such as `KamiHub-1.0.jar`) into the plugins folder under the server directory.
Restart, then enter `/plugins` to check the plugin status.
